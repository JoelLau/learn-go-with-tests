# `range`

Transforms arrays and slices into iterators.
Useful for [iterating](for-loops.md) over [arrays](arrays.md) and [slices.md](slices.md).

## Examples

```go
for index, element := range([]int{50, 100, 150}) {
    // do something
}
```
