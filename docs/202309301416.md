# Modules (Go)

- generated using Go CLI: `go mod init`
- modules are defined in `go.mod`, contain the following info:
    - path to module
    - Go version
    - dependency requirements

Resulting `go.mod`, generated by `go mod init`
```Go
module hello

go 1.20
```
