# Tests (Go)

tests are a special type of function, they must:

1. be placed files with names that ends with `_test.go` in its name (e.g. `hello_test.go`)
2. start with the word `Test`
3. take exactly exactly one argument (`t *testing.T`)
   - must `import "testing"`

to run tests. use the Go CLI (`go test`)
