# switch-case

An alternative to if-else statements that compare an expression (switch)
against multiple results (cases).


## Example(s)

```go
switch person.Gender {
case "Male":
    fmt.Println("duuuudee")
case "Female":
    fmt.Println("gal")
default:
    fmt.Println("whatever you wish to be called")
}
```
